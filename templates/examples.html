Manual

This is a manual to explain how to run our DNA STORAGE app:

First, it is mandatory that you have the following files:

  1. A CSV file containing the design data
  2. A CSV file containing the after alignment data
  3. A CSV file containing the after matching data
  4. A fastq file containing the reads data
IMPORTANT:

You must realize that the correctness of your files is your own responsibility!
Our app will not be able to run efficiently and correctly if there are bugs in the format of your files.
Please remember to double check them before you run the tool.

Go To Tool Page:

Not much to add about this. Just go to the "Tool" page when your done reading this manual.

Preparing Your Files and Data:

You must have the input files ready. Make sure your files are of the correct format. An input file that is meant to be CSV cannot be saved as any other format.
An email must be provided in order for the output file to be sent to you. Please make sure you enter a valid email address.

Uploading Data:

Upload all files to the correct location on the page.Since the files are large, this might take some time.
Once all input has been uploaded, press the "Run Tool" button! You should be redirected to the correct post-run page.
If not, please make sure you followed all steps correctly.
