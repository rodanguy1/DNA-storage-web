Manual

This is a manual to explain how to run our DNA STORAGE app:

First, it is mandatory that you have the following files:

  1. A CSV file containing the design data
  2. A CSV file containing the after alignment data
  3. A CSV file containing the after matching data
  4. A fastq file containing the reads data
IMPORTANT:

You must realize that the correctness of your files is your own responsibility!
Our app will not be able to run efficiently and correctly if there are bugs in the format of your files.
Please remember to double check them before you run the tool.

Uploading Data:

Upload all files to the correct location on the page. The text next to each "Choose File" box should change to the name of your file if file is of the right format

NOTE: Since the files are large, this might take some time.

Once all input has been uploaded, press the "Run Tool" button!


You should be redirected to the correct post-run page.
If not, please make sure you followed all steps correctly.
Remember: This might take some time (can be up to 20 min).





After Run:

An email containing a pdf of your DNA storage analyzation will be sent to the mail given as input.
